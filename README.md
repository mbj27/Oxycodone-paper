# Oxycodone-paper
Code in preparation for manuscript "Investigating the effects of acute oxycodone on brain functional connectivity and brain-based differences in subjective drug response" 
by Maria B. Jelen, Annie Cheng, Robert Kohler, Steven Riley, Lester Rodriguez, Marzieh Babaeianjelodar, Deepa R. Camenga, Mehmet Sofuoglu, Sarah D. Lichenstein, Stephanie M. Gorka, Sarah W. Yip

Corresponding author: Sarah W. Yip

Code correspondence: Maria B. Jelen

Repo contains code for analysis sections:

1. Subjective Drug Response ANOVA (R Markdown)
2. Correlation between connectivity and subjective drug response (R markdown)
3. Seed-based analysis (MATLAB script)
4. Network-based Statistic (NBS output files containing significant edges and model stats - .mat files)
5. Opioid abstinence network comparison (MATLAB script)
